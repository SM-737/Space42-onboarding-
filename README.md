# Space42-onboarding-

Base idea of a glass box orbital twin designed to bridge the gap between the 2 halves of the UAE newly formed space titan. It involves vertical integration of YahSAT and Banyat. By merging data the AI acts as a digital consultant that can explain synergies 

Due to time faced integration errors with lang chain and final deployment, however with iterations the end product can be a game changer platform for a single source of truth approach to prevent AI hallucinations and hence also impact user feedback
